# Pacman备忘

| 操作                         | 命令                     |
| ---------------------------- | ------------------------ |
| 升级所有包                   | pacman -Syu              |
| 安装某个包                   | pacman -S  package_name  |
| 查看已安装的包               | pacman -Qqe package_name |
| 删除某个包, 配置和无用的依赖 | pacman -Rsn package_name |
| 查询本地可用的包             | pacman -Qs package_name  |
| 降级某个包                   | pacman -U package_file   |
